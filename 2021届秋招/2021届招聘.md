特征值分解AX=k*X,求出来特征向量和特征值,A=w(diagm)w-1,,吧A分解成为特征值和特征向量的乘积,矩阵就是线性变换,包含缩放旋转.

奇异值分解可以理解为将A分解为很多个秩为1的矩阵之和,奇异值代表每个秩一矩阵的权重,常用于图像压缩和去噪,(令奇异值较小的那些秩一矩阵直接为0.)
PCA和推荐系统
物理意义:对角矩阵起到线性拉伸的作用,但是奇异值用作(旋转,拉升,投影的一个集合)
代数上表现为:将一组基映射到另一组基.

```
tensorflow优点:
1. 简单的内置高级API

3.难快速修改
pytorch优点:
1.类python的代码,简洁容易修改

3. 容易debug,轻松编辑
4.开源,项目众多
pytorch缺点:
1.可视化工具不如tensorboard,但是现在有了tensorboardx
2.生产部署麻烦

pytorch缺点
```

3. 静态图与动态图的区别
```
tensorflow:静态图,在代码实际运行之前以静态方式生成计算图,优势在于实现并行化或依赖驱动形式调度,让训练更快更加有效.但是不好随意修改,依赖tf的会话机制和placeholder去更更改数据类型.
pytorch:动态图,和python编程方式更为相近,适合初学者.另外debug的时候,pycharm或者pdb可以动态图中使用,但在会话中不能使用这些

```
4. 防止过拟合的方法
```
1. 获取更多数据
2. 数据增强和数据增广
3. 使用合适的模型,不是越深越好
4.early stop
5.正则化(l1,l2正则,在损失函数后面给惩罚,限制模型的)
6. dropout
```
